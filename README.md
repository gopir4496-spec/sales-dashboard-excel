# Sales Dashboard Project

## Overview
Analysis of 1000+ sales transactions across 4 regions in India for 2024.

## Key Insights
- North region had highest sales
- December showed peak revenue (seasonal trend)
- Laptop was the top-selling product

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, SUMIF formulas)
